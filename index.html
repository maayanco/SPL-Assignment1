Main Objective

Correct memory management handling in C++. This assignment simulates a 'cyber-security' attack. Firstly reading xml configuration files and setting up a network of 'computers'. After the building of the network the simulations iterates over the events xml file until termination time, executing events of the types: • Clock-In events (Cyber expert started working) • Hack events (A specific computer is hacked) • Cleaning events (Cyber expert succeded in removing a worm) • Infection events (A worm has infected a certain computer) • Termination events
